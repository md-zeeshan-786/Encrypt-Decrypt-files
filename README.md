# Encrypt-Decrypt-files

Steps to use app:-

SignUp & SignIn page,
1] SignUp using user id & strong passwrod.
2] SignIn using that same user id & password.

Encryption,
1] browse your files/enter path location of your files to encrpt.
2] select file & open > enter strong passwrod to encrypt > click on Encrypt.
3] Encrypted a ".enc" file will be generated in that location where you selected that file to encrypt.
4] After opening the .enc file the data will now be converted into cypher text.

Decryption,
1] browse your files/enter path location of your .enc file to dencrpt.
2] select file & open > enter passwrod at encryption time to dencrypt > click on decrypt.
3] Decrypted a ".decry" file will be generated in that location where you selected that file at encryption time.
4] After opening the .decry file the data will now be converted cypher into original text.
